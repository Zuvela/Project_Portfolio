## Key Projects

### Autonomous Robot-Based Testing System
Development of a fully autonomous testing system using computer vision, AI, and a robot arm.

- Computer vision-based detection of interaction points  
- AI-driven generation of robot execution logic  
- Integration of perception, decision-making, and actuation  
- Focus on real-world deployment and adaptability  

---

### Hand Gesture Recognition using CNNs
Machine learning project for real-time gesture classification using PyTorch and transfer learning.

- CNN model based on pretrained ResNet18  
- Achieved 97.9% validation accuracy  
- Implemented data augmentation, label smoothing, and weighted sampling  
- Designed to handle noisy and non-relevant inputs using a rejection class  

---

### Roll-to-Roll AI Monitoring System
Automated system for droplet detection and monitoring in material deposition processes.

- Embedded control using Arduino  
- Computer vision for droplet detection  
- Real-time feedback integration  
- Custom experimental setup and validation  

---

### Joystick Test System – Danfoss Power Solutions
Development of automated testing systems for industrial joystick validation.

- Robot-assisted testing for repeatability  
- CAN-based data acquisition and analysis  
- Sensor integration and validation workflows  

---

### Ball Joint Sensor Integration
Bachelor thesis focused on extracting rotational data from a constrained mechanical system.

- Sensor integration and signal processing  
- Indirect state estimation  
- Experimental validation and calibration
